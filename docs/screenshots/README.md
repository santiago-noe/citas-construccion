Guía rápida para capturas de funcionamiento

- Archivo de salida de la app: `docs/screenshots/app_output.txt`

Cómo tomar capturas en Windows:
- Pantalla completa: pulsa `PrtScn` y pega en tu editor (Paint/Word).
- Captura de ventana activa: pulsa `Alt + PrtScn`.
- Recorte parcial (recomendado): `Win + Shift + S` y selecciona área; la imagen queda en el portapapeles.
- Herramienta: abre "Recortes y anotaciones" (Snip & Sketch) para guardar directamente.

Sugerencia para el informe:
- Incluye `app_output.txt` como evidencia textual.
- Añade capturas de pantalla (PNG/JPG) en la misma carpeta y referencia los archivos en tu informe.

Si quieres, puedo intentar generar capturas de pantalla automáticas desde PowerShell (requiere permisos GUI). Dime si quieres que lo haga.
